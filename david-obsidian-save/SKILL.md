# Obsidian Save 스킬

## 목적

현재 Claude와 나눈 대화를 요약하여 사용자의 Obsidian vault에 마크다운 노트로 저장한다.
나중에 다시 찾아볼 수 있도록 잘 구조화된 노트를 만드는 것이 핵심이다.

## Vault 경로

```
~/Library/CloudStorage/GoogleDrive-dochoul@gmail.com/내 드라이브/Obsidian/
```

## 실행 단계

### 1. 주제(topic) 파악

대화 전체를 훑어보고 핵심 주제를 2~4개 단어로 요약한다.
파일명에 들어가므로 한국어로 간결하게 (예: `Claude Code 활용법`, `옵시디언 연동 방법`).

### 2. 파일명 결정

형식: `YYYY-MM-DD 주제.md`

예시:
- `2026-06-26 Claude와 옵시디언 연동.md`
- `2026-06-26 Python 데이터 분석 방법.md`

오늘 날짜를 사용한다. 같은 날짜에 같은 주제 파일이 이미 있으면 `(2)`, `(3)` 등을 붙인다.

### 3. 노트 내용 작성

아래 형식을 따른다:

```markdown
---
date: YYYY-MM-DD
tags:
  - claude
  - (대화 주제와 관련된 태그 1~3개)
---

# 주제

## 요약

(대화 전체를 3~5문장으로 핵심만 요약. 핵심 결론이나 배운 점 위주로.)

## 주요 내용

(대화에서 다룬 내용을 소제목별로 정리. 불릿 포인트 활용.)

## 참고 / 메모

(코드 스니펫, 링크, 추가로 기억해둘 사항 등. 없으면 이 섹션 생략.)
```

### 4. 파일 저장

Write 도구를 사용해 vault 경로에 저장한다.

### 5. 완료 알림

저장이 완료되면 다음을 알린다:
- 저장된 파일 경로
- 요약 내용 (미리보기)

## 작성 가이드라인

- **언어**: 노트 본문은 한국어로 작성한다. (코드나 고유명사는 원어 유지)
- **간결함**: 장황하게 모든 대화를 옮기지 말고, 핵심만 추출한다.
- **태그**: 나중에 검색에 유용하도록 실질적인 태그를 붙인다. (예: `claude`, `개발`, `python`, `obsidian`)
- **실용성**: 나중에 이 노트를 다시 읽을 때 바로 이해할 수 있도록 맥락을 충분히 담는다.

# Google Drive → Obsidian 유튜브 링크 연동

## 목적

Google Drive Obsidian 폴더에 저장된 `.txt` 파일에서 유튜브 링크를 읽어
`유튜브 나중에 보기.md`에 정리하고 `.txt` 파일을 삭제한다.

## 실행 단계

### 1. .txt 파일 확인

vault 경로에서 `.txt` 파일을 찾는다. 없으면 "처리할 링크가 없습니다"라고 알리고 종료.

### 2. 유튜브 링크 추출 및 제목 가져오기

각 링크를 WebFetch로 접근해 영상 제목을 가져온다.
제목을 가져오지 못한 경우 링크만 추가한다.

### 3. 유튜브 나중에 보기.md 업데이트

파일이 없으면 새로 만든다. 추가 형식:

```markdown
## 영상 제목

- 🔗 유튜브 링크
```

이미 있는 링크는 중복 추가하지 않는다.

### 4. .txt 파일 삭제

처리 완료된 `.txt` 파일을 삭제한다.

### 5. 완료 알림

추가된 영상 목록을 알린다.
